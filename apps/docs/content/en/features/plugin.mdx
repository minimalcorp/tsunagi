# Tsunagi Plugin

Tsunagi ships a Claude Code plugin (`tsunagi-plugin`) that is automatically
installed when Tsunagi starts and uninstalled when it stops. This plugin is
what makes Tsunagi's deep integration with Claude possible: it bundles an
**MCP server**, a **skill**, a **curator agent**, and the **hook handlers**
that stream Claude's activity into the Tsunagi UI in real time.

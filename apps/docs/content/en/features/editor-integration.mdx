# Editor Integration

Tsunagi includes a built-in **Monaco editor** modal for opening and
editing files without leaving the UI. This is useful for:

- Editing prompts as Markdown with proper syntax highlighting
- Quickly tweaking files inside a worktree
- Reviewing diffs before asking Claude to continue

## Triggering Monaco from Claude

Tsunagi sets `EDITOR` in the shell environment of every Claude session to
a small helper script. When Claude (or you) runs a command like `git
commit` that opens an editor, the helper opens a Monaco modal in the
Tsunagi UI instead of a terminal-based editor.

You can also press a key shortcut from the terminal view to open Monaco
on an arbitrary file path.

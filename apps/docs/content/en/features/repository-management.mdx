# Repository Management

Tsunagi can manage multiple repositories across multiple GitHub
organizations from a single UI. Each registered repository is cloned as a
bare repository under `~/.tsunagi/workspaces/<owner>/<repo>/.bare/` and
serves as the base from which task worktrees are spawned.

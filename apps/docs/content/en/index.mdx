# Tsunagi

**Tsunagi (繋ぎ)** は、複数 GitHub 組織のプロジェクトをローカルで統合管理し、
Claude AI による開発タスクを効率化する Web UI ツールです。

## 主な特徴

- **統合管理**: 複数 GitHub organization のプロジェクトを 1 つの UI で管理
- **Git Worktree 活用**: タスクごとに独立した作業環境を自動構築
- **Claude Agent 統合**: タブ型セッション管理でリアルタイムに AI 実行を可視化・制御
- **ローカル完結**: 全てのデータと作業がローカル環境で完結

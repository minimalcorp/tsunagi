# Installation

## 前提条件

- **Node.js 20+**
- **Git 2.42+**（空リポジトリ対応で `git worktree add --orphan` を使用）
- **Claude Code CLI**（`claude` コマンドが PATH 上に必要）
- **macOS または Linux**（Windows は非サポート）
- **Anthropic API Key** または **Claude Code OAuth Token**

## npm で起動（推奨）

```bash
npx @minimalcorp/tsunagi
```

または、グローバルインストール:

```bash
npm install -g @minimalcorp/tsunagi
tsunagi
```

起動後、ブラウザで [http://localhost:2791](http://localhost:2791) を開きます。

## 起動時に起こること

Tsunagi は起動時に以下を自動で行います:

1. `~/.tsunagi/state/tsunagi.db` に SQLite データベースを初期化（存在しない場合）
2. Prisma マイグレーションを適用
3. `~/.tsunagi/state/tsunagi.lock` に PID を書き込み、単一インスタンスをロック
4. Claude Code の marketplace と plugin をクリーンインストール
   - 既存の tsunagi plugin がある場合は一旦 uninstall してから再インストール
   - `claude` CLI 経由でのみ操作し、`~/.claude/settings.json` を直接編集することはありません
5. Fastify サーバー（ポート 2792）と Next.js サーバー（ポート 2791）を起動
6. 終了時に plugin と lock ファイルをクリーンアップ

## 対応 OS

Tsunagi は **macOS と Linux** のみをサポートします。Windows ユーザーは WSL2
経由で動作する可能性がありますが、動作保証はありません。

## データ保存場所

```
~/.tsunagi/
├── state/
│   ├── tsunagi.db            # SQLite データベース
│   └── tsunagi.lock          # 単一インスタンスロック
├── backups/                  # DB バックアップ
└── workspaces/               # Git worktrees
```

環境変数 `TSUNAGI_DATA_DIR` で保存先を変更できます。

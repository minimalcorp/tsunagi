# Tsunagi プラグイン

Tsunagi は Claude Code プラグイン (`tsunagi-plugin`) をバンドルしており、Tsunagi の起動時に自動 install され、終了時に自動 uninstall されます。このプラグインこそが Tsunagi と Claude の深い統合を実現している中核です: **MCP server**、**skill**、**curator agent**、そして Claude の活動を Tsunagi UI にリアルタイムで流し込む **hook handler** をバンドルしています。

# リポジトリ管理

Tsunagi は複数の GitHub organization にまたがる複数のリポジトリを 1 つの UI から管理できます。登録されたリポジトリは `~/.tsunagi/workspaces/<owner>/<repo>/.bare/` に bare repository としてクローンされ、タスク用 worktree を派生させるベースとして機能します。
